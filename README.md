# 📦 Inventory Management System

A professional **Inventory Management System** developed using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**.  
This system helps businesses manage their inventory, sales, users, and customer orders efficiently.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Secure Admin Login
- Dashboard with analytics:
  - Total Products
  - Total Sales
  - Total Users
  - Pending Requests
- Add / Update / Delete Products
- Manage Users
- View Sales Reports
- Approve Product Requests
- Create Orders for Suppliers
- Index Creation / Deletion

---

### 👤 User Panel
- Secure User Login
- Dashboard
- View Available Products
- Sell Products
- Request New Products
- View Sales History

---

### 🛍 Customer Panel
- Secure Customer Login
- Dashboard
- Shop Products
- Place Orders
- View Order History

---

## 🔐 Security Features
- Password Hashing
- CSRF Protection
- SQL Injection Prevention using Prepared Statements
- Input Validation & Sanitization
- Session-based Authentication
